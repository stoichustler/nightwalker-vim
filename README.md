### Nightwalker

Install `nightwalker` without plugin manager.

```sh
git clone https://github.com/stoichustler/nightwalker-vim.git ~/.vim/pack/plugins/start/nightwalker
```

add this to your .vimrc

```vim
set termguicolors

let g:nightwalker_style = 'night' " available: night, storm
let g:nightwalker_enable_italic = 1

colorscheme nightwalker
```
