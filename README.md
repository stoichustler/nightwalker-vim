### Nightwalker

add this to your .vimrc

```vim
set termguicolors

let g:tokyonight_style = 'night' " available: night, storm
let g:tokyonight_enable_italic = 1

colorscheme tokyonight
```

```sh
git clone https://github.com/stoichustler/nightwalker-vim.git ~/.vim/pack/plugins/start/nightwalker
```
